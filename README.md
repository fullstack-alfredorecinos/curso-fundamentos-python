### Comandos básicos de Python
-python --version: Muestra la versión de Python instalada.
-python: Inicia el intérprete de Python.
- exit(): Sale del intérprete de Python.
- help(): Muestra la ayuda interactiva.
- dir(): Muestra los atributos y métodos de un objeto.
- `print()`: Imprime en la consola.
- `input()`: Solicita entrada del usuario.
- `len()`: Devuelve la longitud de un objeto.
- `type()`: Devuelve el tipo de dato de un objeto.
- `int()`, `float()`, `str()`: Convertidores de tipos de datos.
- `if`, `elif`, `else`: Estructuras de control condicional.
- `for`, `while`: Estructuras de control de bucles.
- `def`: Define una función.
- `class`: Define una clase.
- `import`: Importa módulos y bibliotecas.
